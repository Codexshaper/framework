== Description ==

CodexShaper Framework is one-stop wordpress solution that supercharges your website design with over 90 custom widgets. 
Tailored for both beginners and experts, it offers a user-friendly drag-and-drop interface, allowing you to create stunning, pixel-perfect websites 
without any coding. With unique features like advanced controls and powerful modules, CodexShaper Framework empowers you to easily build and customize your 
online presence. Elevate your web development game and unlock the full potential of your WordPress experience!

== Frequently Asked Questions ==

= How do I install CodexShaper Framework? =

Go to your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter CodexShaper Framework and choose CodexShaper Framework.

Press install -> After installation, click Activate.

Note: Make sure your are installed Elementor plugin to use CodexShaper Framework.

= Can I build any kinds of design using CodexShaper Framework? =

You can build any kinds of design using Elementor and CodexShaper Framework.

= Do I need to know coding to use CodexShaper Framework? =

Nope, you don't need to know any coking skills.

= Do I need to know design? =

Nope, You don't need but if you have good design sense then you will get extra outcome.
